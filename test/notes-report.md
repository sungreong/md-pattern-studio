## 주요 작업 {#work .two-column}

### 온보딩 개선
- 가입 단계 수를 5단계에서 3단계로 축소했다.
- 필수 입력 항목 설명을 실제 사용자 질문 기반으로 수정했다.
- 인증 오류 메시지를 원인 중심 문장으로 정리했다.

### 리포트 생성 개선
- 중복 조회 쿼리를 제거해 DB 부하를 줄였다.
- 집계 순서를 통일해 결과 편차를 낮췄다.
- 실패 시 재시도 로직을 공통 모듈로 정리했다.
